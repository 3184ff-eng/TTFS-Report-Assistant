# Official TTFS Fire Report Form Template

The unchanged official TTFS Fire Report Form is stored here as:

```text
public/templates/ttfs-fire-report-form.pdf
```

The application print/PDF layer uses rendered page images from that immutable PDF:

```text
public/templates/ttfs-fire-report-form-page-1.png
public/templates/ttfs-fire-report-form-page-2.png
```

Generated content is overlaid into the corresponding form areas and overflow continues onto appendix pages.
